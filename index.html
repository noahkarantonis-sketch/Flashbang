Setting up — 4 quick steps

  1. Download the file above (a .zip).
  2. Unzip it — right-click the downloaded file → Extract All.
  3. Open the unzipped folder and double-click Flashbang.exe.
    First launch only: if Windows shows a blue "Windows protected your PC" box, click More info → Run anyway. (It's safe — the app just isn't code-signed yet.)
  4. Sign up with your email and start making cards.
